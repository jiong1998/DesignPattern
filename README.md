# DesignPattern

主流的**设计模式**有23种，主要编写了以下几种模式的代码，其他方法等遇到了再学习：
- **创造型模式**：用来描述 “如何创建对象”，它的主要特点是 “将对象的创建和使用分离”。包括单例、原型、工厂方法、抽象工厂和建造者 5 种模式。
	- 单例模式
	- 工厂方法模式
	- 建造者模式
- **结构型模式**：用来描述如何将多个类或对象按照某种布局组合成更大的结构。包括代理、适配器、桥接、装饰、外观、享元和组合 7 种模式。
	- 代理模式
- **行为型模式**：用来识别多个类之间的常用交流模式以及如何分配职责
	- 策略模式
	- 模版方法模式
